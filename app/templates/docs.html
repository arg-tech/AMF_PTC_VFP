Argument Mining Framework: Proposition Type Classifier module (AMF-PTC-VPF).

Functionality: Predict the type of the argumentative propositions. The module classifies propositions into the types of Value, Policy, and Fact.

Expected Input: xAIF file containing propositional argumentative nodes (I).

Expected Output: xAIF file with the "propositionClassifier" key containing the list of proposition (I) nodes with one of the three types (i.e., Value, Policy, and Fact) assigned to them.
